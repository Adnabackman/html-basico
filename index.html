Aula de HTML e CSS pelo programa Start na Capgemini.

Acesse o site aqui.

Primeira Aula

Na primeira aula estamos aprendendo sobre regras básicas do CSS, blocos, propriedades, valores. Esta aula focou na parte teórica do html e sua estrutura.

Segunda Aula

Na segunda aula estamos aprendendo sobre estilização dentro do HTML, com classes, id's, comentários e etc. Esta aula focou na parte prática do html e no style do index.

Terceira Aula

Na terceira aula estamos aprendendo sobre estilização no CSS, como rgba e cores Hexadecimais. Esta aula focou na parte prática do html e no style do css.

Quarta Aula

Na quarta aula estamos aprendendo sobre estilização no CSS, com background-color, backgroun-image, background-repeat. Esta aula focou na parte prática do html e no style do css.

Quinta Aula

Na quinta aula estamos aprendendo sobre estilização no CSS, com Font-size Font-family, Font-face, Font-style, Font-weight e Font-variant.. Esta aula focou na parte prática do html e no style do css.
